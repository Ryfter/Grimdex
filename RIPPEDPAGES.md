# Ripped pages — removals ledger

Anything removed from `GRIMDEX.md` or a playbook lands here: what was torn out, why,
the date, and what (if anything) replaced it. Append-only, newest on top. Removals are
**always human-gated** — the sweep proposes, the owner rips. The sweep also checks new
promotion candidates against this file so trimmed ideas don't sneak back in.

Entry format:

```
## YYYY-MM-DD — <short title of the removed rule>
**Removed from:** GRIMDEX.md | playbooks/<name>.md
**The rule was:** <verbatim or faithful summary>
**Why removed:** <reasoning — this is self-training data>
**Replaced by:** <new rule / nothing>
**Ruled by:** <owner> (sweep proposed YYYY-MM-DD) | <owner> directly
```
